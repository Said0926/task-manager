# Task Manager (ToDo App)

Простой ToDo проект на Django, созданный для портфолио.  
Позволяет пользователям регистрироваться, авторизоваться и управлять своими задачами.

---

## 🔹 Функционал

- ✅ Регистрация и авторизация пользователей (login/logout/register)  
- ✅ Создание, редактирование и удаление задач  
- ✅ Привязка задач к конкретному пользователю  
- ✅ Список задач с фильтрацией по пользователю  
- ✅ Статусы задач: "Не сделано", "В работе", "Сделано"  
- ✅ Базовая защита с CSRF  
- ✅ Шаблоны с использованием Bootstrap и базового `base.html`  

---

## 🔹 Технологии

- Python 3.10+  
- Django 5  
- HTML/CSS + Bootstrap 5  
- SQLite (база данных по умолчанию)  
- Git & GitHub  

---

## 🔹 Установка и запуск

1. Клонировать репозиторий:

```bash
git clone https://github.com/Said0926/task-manager.git
cd task-manager

Создать виртуальное окружение и активировать его:

python -m venv venv
source venv/bin/activate   # macOS / Linux
venv\Scripts\activate      # Windows

Установить зависимости:
pip install -r requirements.txt

Сделать миграции:
python manage.py makemigrations
python manage.py migrate

Запустить сервер:
python manage.py runserver

Перейти в браузере:
http://127.0.0.1:8000/

🔹 Использование
Регистрация: /register/
Авторизация: /login/
Создание новой задачи: /create-task/
Редактирование/удаление задач: ссылки рядом с каждой задачей (доступны только для своих задач)

🔹 Особенности кода
Используется LoginRequiredMixin для защиты приватных страниц
Привязка задач к пользователю через ForeignKey
Шаблоны основаны на base.html для единообразного дизайна
Удобный navbar с проверкой авторизации пользователя
